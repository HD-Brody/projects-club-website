Backend Flask app (starter)

- Create a Python virtual environment inside `backend/` (not checked into git): `python -m venv .venv`
- Activate it: .\.venv\Scripts\Activate.ps1
- Install requirements: `pip install -r requirements.txt`
- Create .env file in backend and configure environment variables based on .env.example
- Run: `python run.py`

API will be available at http://localhost:5000 and CORS is configured to allow the Vite frontend at http://localhost:5173.
